# 功能介绍

## 纯洁

LantaWrt非常的纯洁，只保留了最基本的翻墙功能

## 轻量

LantaWrt的资源占用非常低

## 美观

自带[Argon](https://github.com/jerrykuku/luci-theme-argon)主题，高雅而美观

## 更改

- 增加了[Hello World](https://github.com/jerrykuku/luci-app-vssr)（默认开启SSR、SS、V2ray、VLESS、Trojan的支持）
- 默认开启iPv6Helper
